# 📚 Library Management System

A simple **Library Management System** designed to manage books and basic library records using a web-based interface.

🔗 **Live Demo:**  
👉 [ https://mfathumsachcha-netizen.github.io/library_management-Demo/](https://proportial-duck.github.io/library_management-Demo/)


## ✨ Features
- Add, view, update, and delete books
- Search and manage library records
- Simple and user-friendly interface
- Dynamic data handling with database support

## 🛠️ Technologies
HTML • CSS • JavaScript  
PHP • Database (MySQL)

## 🚀 How to Run
1. Install XAMPP / WAMP / MAMP  
2. Place the project folder in `htdocs`  
3. Configure database credentials  
4. Start Apache and open in browser

## 🎯 Purpose
Built as an academic project to practice **CRUD operations** and backend integration.
